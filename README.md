# summary
- hooks为函数组件引入state和生命周期函数 , 从而使函数组件可以完全替代类组件;
- 相比于类组件中的`this`以及`bind` , hooks让组件变得更精简,更容易使用;
- hooks使得组件间状态逻辑复用更容易更灵活(自定义hook)
